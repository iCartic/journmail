Teach yourself consistency by keeping a journal and writing one sentence per day in it.

## Stack

* Ruby
* PostgreSQL

## Development

Spin up a development server

```shell
$ rackup
```

Run tests

```shell
$ bundle install
$ rake
```

Set up database

```shell
$ rake db:create
$ rake db:migrate
```
